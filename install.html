Установка XBT трекера для DLE+XBT Help

Установка дополнительных пакетов

Инструкция с 0-web.ru

apt-get install cmake g++ libboost-date-time-dev libboost-dev libboost-filesystem-dev libboost-program-options-dev libboost-regex-dev libboost-serialization-dev libmysqlclient15-dev make zlib1g-dev
yum install boost-devel gcc-c++ mysql-devel

Если ошибка - Processing Conflict mysql55 conflicts mysql: Решение: Вместо mysql-devel установить mysql55-devel:

yum install boost-devel gcc-c++ mysql55-devel subversion

Если ошибка - Error Package mysql-devel-5.1.73-3.el6_5.x86_64 (updates) Requires mysql = 5.1.73-3.el6_5 Решение:

cd /etc/yum.repos.d wget http://rpms.famillecollet.com/enterprise/remi.repo yum --enablerepo=remi install mysql-devel

Обновляем порты:

portsnap fetch portsnap update

Внимание! На время установки рекомендую отключить apache и mysql:

/usr/local/etc/rc.d/apache22 stop /usr/local/etc/rc.d/mysql-server stop

Далее:

make -C /usr/ports/devel/subversion install clean make -C /usr/ports/devel/boost-jam install clean make -C /usr/ports/devel/boost-libs install clean make -C /usr/ports/lang/gcc48 install clean rehash

Инструкция из сети

apt-get install cmake default-libmysqlclient-dev g++ git libboost-dev make zlib1g-dev
dnf install boost-devel cmake gcc-c++ git make mysql-devel systemd-devel

Создать ярлык x64 (/usr/lib64) библиотек для x86 (/usr/lib)

ln -s /usr/lib64/mysql/libmysqlclient.so /usr/lib/libmysqlclient.so ln -s /usr/lib64/libsystemd.so.0 /usr/lib/libsystemd.so.0
